Index of module types


C
Client [Cohttp_lwt]
The Client module implements non-pipelined single HTTP client calls.

I
IO [Cohttp_lwt]
The IO module is specialized for the Lwt monad.

N
Net [Cohttp_lwt]
The Net module type defines how to connect to a remote node and close the resulting channels to clean up.

R
Request [Cohttp_lwt]
The Request module combines the Cohttp.Request module with the IO functions, to have them conveniently in one place.
Response [Cohttp_lwt]
The Response module combines the Cohttp.Request module with the IO functions, to have them conveniently in one place.

S
Server [Cohttp_lwt]
The Server module implements a pipelined HTTP/1.1 server.
